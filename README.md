# Sistema de Gestión de Inventario para el Conservatorio Nacional de Música

## 📌 Descripción del Proyecto
Este proyecto implementa un **sistema de gestión de inventario** para el almacén de instrumentos y suministros del Conservatorio Nacional de Música. Su objetivo es mejorar la organización, el control y la disponibilidad de los recursos, eliminando el extravío de instrumentos y reduciendo las ineficiencias operativas.

## ❓ Problema de Estudio
El Conservatorio enfrenta dificultades en la gestión del inventario debido a:
-  Extravío frecuente de instrumentos y suministros.
-  Falta de información actualizada sobre el inventario.
-  Retrasos en la asignación de instrumentos.
-  Procesos manuales y herramientas obsoletas que generan ineficiencias.

Este sistema propone una solución tecnológica adaptada a las necesidades específicas del Conservatorio.

## ✅ Justificación
La implementación de este sistema permitirá:
-  Reducir la pérdida de instrumentos y suministros.
-  Agilizar el acceso a información actualizada del inventario.
-  Mejorar la asignación y devolución de recursos.
-  Optimizar los procesos operativos del Conservatorio.

## 🛠 Tecnologías Utilizadas
- **Lenguaje de programación**: Python
- **Base de datos**: SQLite
- **Interfaz gráfica**: PyQt
- **Metodología de desarrollo**: Scrum
